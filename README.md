# portfolio-website-

📌 Project Overview
The portfolio website serves as an online resume and professional showcase. It includes sections such as About Me, Skills,
Resume, Projects, and Contact, allowing visitors to explore my experience and technical capabilities. 
The website is designed with a clean user interface and responsive layout so it works smoothly across different devices.

The platform highlights my work in Artificial Intelligence, Machine Learning, Data Analytics, and Web Development, and provides links to my GitHub projects and LinkedIn profile for further exploration.

✨ Features
🏠 Hero Section
Introduction with name, role, and social links (GitHub and LinkedIn).
👨‍💻 About Section
Personal background, career interests, and technical expertise.
🧠 Skills Section
Displays technical skills such as Python, Machine Learning, SQL, Tableau, and Web Development with progress bars.
📄 Resume Section
Educational background and career information with an option to download the CV.
💻 Projects Section
Showcases major projects including:
Fake Job Detection (NLP)
AI Object Detection
Netflix Tableau Dashboard
Weather Check Web Application.
📬 Contact Section
Contact information including phone, email, LinkedIn, and a contact form.
📱 Responsive Design
Mobile-friendly navigation with a responsive menu and smooth scrolling.

🛠️ Technologies Used
HTML5 – Website structure
CSS3 – Styling and responsive design
JavaScript – Interactive features such as menu toggle, smooth scrolling, and form handling
